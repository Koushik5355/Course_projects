Digital Attendance System (C Language)
======================================

A simple console-based attendance management system written in C that allows you to:

- Register students by their IDs
- Mark attendance
- View present and absent students
- Display all registered student IDs

--------------------------
Features
--------------------------
- Register up to 100 students with unique IDs
- Mark attendance with validation to avoid duplicate entries
- Display:
  - Present students
  - Absent students
  - All registered students
- Menu-driven interface for easy use

--------------------------
How It Works
--------------------------
1. Registration Phase:
   - User is prompted to input the number of students to register and their IDs.
2. Menu Options:
   === Attendance Menu ===
   1. Mark Attendance
   2. Show Present Students
   3. Show Absent Students
   4. Show All Registered Students
   5. Exit

--------------------------
How to Run
--------------------------
1. Save the code in a file, e.g., attendance.c
2. Compile and run using a C compiler:

   gcc attendance.c -o attendance
   ./attendance

--------------------------
File Structure
--------------------------
- attendance.c – Main source file containing the code logic
- README.txt – Project documentation

--------------------------

- Menu-Driven Loops
- Input Validation
- Functions and Structures
